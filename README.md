# zbn-ops-dashboard

Dashboard operativa Zetabytenexus per follow-up, portfolio e flag commerciali.

## Principi
- I dati operativi veri restano nei repo progetto e in `opencode-config`.
- Questa repo pubblica una vista statica e leggibile via GitHub Pages.
- I file JSON in `app/data/` sono il layer dati minimo condiviso tra OpenCode e dashboard.

## Avvio
Apri `app/index.html` in locale oppure pubblica tramite GitHub Pages con GitHub Actions.
